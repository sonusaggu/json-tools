Stringify Options

Stringify Rules

  • Converts JavaScript objects to JSON strings
  • Handles all valid JSON data types (strings, numbers, arrays, objects, booleans, null)
  • Properly escapes special characters when enabled
  • Optionally formats with indentation (spaces or tabs)
  • Preserves data types during conversion

Common Issues

IssueExampleSolution
Circular referenceslet a = {}; a.b = a;Remove circular references or use JSON.stringify() with a replacer function
Non-JSON values{date: new Date()}Convert to string or use replacer function
Malformed JSON{key: undefined}Ensure all values are JSON-compatible (remove undefined values)
Large objects{data: [...]} // 10MB+Process in chunks or use streaming JSON parser

Pro Tips

  • Use pretty printing for debugging and development
  • For APIs, use compact format to reduce payload size
  • Handle dates/functions with custom replacer functions
  • Validate JSON before stringifying to catch errors early
  • For large datasets, consider processing in batches
